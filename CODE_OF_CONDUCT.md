# Code Of Conduct

This project expects professional, respectful collaboration.

## Expected Behavior

- Be clear, direct, and constructive.
- Discuss technical tradeoffs with evidence.
- Assume good intent while still reviewing carefully.
- Keep feedback focused on the work.
- Respect privacy, credentials, inbox data, and user contact data.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private credentials, inbox data, customer/contact data, or secrets.
- Pressuring maintainers to merge unsafe changes.
- Submitting code that bypasses send policy, confirmation, or audit controls without explicit review.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or contributions that violate this code of conduct. Repeated or severe violations may result in a block from participating in the project.
